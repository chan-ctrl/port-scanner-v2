# Port Scanner V2

A cybersecurity project built using Python.

## What this project does

This tool scans a target IP address and checks whether ports in a user-specified range are open.

## Features

- User enters target IP address
- User chooses starting and ending ports
- Scans ports within the specified range
- Displays open ports
- Uses Python socket programming

## Technologies Used

- Python 3
- Socket Library

## How to Run

python port_scanner_v2.py

## Example

Enter IP Address: 127.0.0.1
Enter Starting Port: 20
Enter Ending Port: 100
